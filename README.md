# Vue 挖矿小游戏
本项目基于Vue.js开发, 项目想法来源于[广树](https://github.com/eeg1412)的[维基萌抽卡 - 星星矿场](https://github.com/eeg1412/wikimoeCardByNodeJS)

# 项目预览
![预览图](https://s1.locimg.com/2024/05/27/457014f59c8b0.png)

# 安装依赖
npm i

# 编译项目
npm build

# 运行项目
npm run serve

# 技术栈
Vue, Vuex, Element-ui

# 在线体验
https://setube.github.io/vue-mining
